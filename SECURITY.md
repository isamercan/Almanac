# Security Policy

## Supported Versions
The latest released minor version receives security fixes.

## Reporting a Vulnerability
Please **do not** open a public issue for security reports. Instead, use GitHub's private
[Security Advisories](https://docs.github.com/en/code-security/security-advisories) ("Report a
vulnerability") on this repository, or email the maintainers.

We aim to acknowledge reports within a few business days and to ship a fix or mitigation promptly,
crediting the reporter unless anonymity is requested.
